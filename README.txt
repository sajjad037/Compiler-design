Instruction How to run the program for Assignment 1.
1) Program is consist for several different packages, each package contains classes related to it domain.
2) Package 'com.project.compiler.driver' contains class 'CodeGenerationDriver.java' which main method to execute the program. 
	a) It is the driver class.
	b) All test cases are defined there in 'testCases' String[] variable.
	c) Result to each test case will be save in test case directory. 
	b) After running the test case, two files Ast.txt, SymbolTable.txt, MoonCode.m and errors.txt will be saved in test case directory.
3) Package 'com.project.compiler.config' contains the configuration or static/hard code values used in the program, we can change
	these values as per requirement.
